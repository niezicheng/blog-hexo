工具库

介绍

这里是我平时使用的一些工具库，包括一些常用的工具库、在线工具、图片视频工具等，方便查找和使用。

图片 & 视频

高清图片

  • unsplash - 一个高清图片库，可以查看各种高清图片，方便使用。

图片在线压缩

  • TinyPNG - 一个在线图片压缩工具，可以将图片压缩到更小的尺寸，减少图片的大小，提高页面加载速度。
  • Squoosh - 一个由 Google 开发的在线图片压缩工具，可以对图片进行多种格式的压缩，支持 WebP、JPEG、PNG 等格式。

代码生成图片

  • Carbon - 一个代码生成图片的工具，可以将代码生成图片，方便分享到社交平台。
  • ray.so - 一个代码生成图片的工具，可以将代码生成图片，支持多种主题和语言。

视频·图片转换

  • ezgif - 一个视频图片转换工具，可以将视频转换为图片，也可以将图片转换为视频。
  • gifrun - 一个在线 GIF 图片编辑工具，可以对 GIF 图片进行编辑，添加文字、裁剪、压缩等操作。

无背景图片·资源

  • pngdirs - 一个无背景图片资源库，可以查看各种无背景图片，方便使用。
  • remove.bg - 一个在线去除图片背景的工具，可以将图片背景去除，生成无背景图片。

本地图片·在线链接

  • sm.ms - 一个免费的图片上传服务，可以上传本地图片，获取图片在线链接，永久保存图片。
  • imgbb - 一个免费的图片上传服务，可以上传本地图片，获取图片在线链接，永久保存图片。
  • picgo - 一个开源的图片上传工具，可以上传本地图片，获取图片在线链接，支持多种图床。

数据格式化

JSON 格式化

  • Json.cn - 一个在线 JSON 格式化工具，可以将 JSON 数据格式化输出，方便查看和调试。

交互·设计动效

表情符号

  • EmojiALL - 一个表情符号大全，可以查看各种表情符号，方便复制到文档中使用。

css3 动画

  • css3-animation - 一个 css3 动画代码片段集合，并且包含片段代码，直接 cv 就可以了

动效·交互组件·uiverse

  • uiverse - 一个动效交互组件库，可以查看各种动效交互组件，方便使用。

模板

封面模板·canva

  • canva - 一个封面模板库，可以查看各种封面模板，方便使用。

网站工具集

神器集 - 一个网站工具集，可以查看各种网站工具，方便使用。

字体·图标库

  • Google Fonts - 一个免费开源在线字体库，可以查看各种字体，方便使用。
  • Font Awesome - 一个免费开源图标字体库，可以查看各种图标字体，方便使用。
  • iconfont - 一个阿里巴巴图标库，可以查看各种图标字体，方便使用。
